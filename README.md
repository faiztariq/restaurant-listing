# Restaurant Listing
Repository for a POC on Restaurant Listing App

## Setup
API (JSON Server)

```
cd api
yarn
```

UI (NextJS Table UI)

```
cd ui
yarn
```

## Running

API

```
cd api
npx json-server db.json
```

UI

```
cd ui
yarn dev
```

- Open a web browser and go to `http://localhost:4200`
- Hit the `Refresh` Button for the UI to post new Data
